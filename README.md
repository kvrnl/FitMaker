# FitMaker

**Create custom FiveM clothing — no Blender, no OpenIV, no CodeWalker. By [KVRNL](https://kvrnl.io).**

Pick a base garment, restyle it (colours, patterns, your own logos placed on the clothing), preview
it in 3D on the freemode ped, and export a drop-in resource folder that streams straight onto your
server. Male and female freemode supported.

### Download
Get it free at **[kvrnl.io/products/fitmaker](https://kvrnl.io/products/fitmaker/)** — create a free
account, claim your license key, and install. Windows.

> FitMaker is not code-signed yet, so Windows SmartScreen may warn on first run.
> Click **More info -> Run anyway**.
